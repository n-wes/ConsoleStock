### Dependencies
Docker

### Installation Instructions
In the root project directory:  
> docker-compose build  

### Startup Instructions
In the root project directory:  
> docker-compose up  

### Shutdown Instructions
In the root project directory:  
> docker-compose down  
>
> docker volume prune
